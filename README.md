# 💰 Smart Expense Tracker
A modern desktop application built using Python to efficiently track and manage daily expenses with a clean graphical user interface.

## 🚀 Features
* Add, delete, and manage daily expenses
* Filter expenses by category
* Real-time expense tracking in a structured table
* Category-wise data visualization using bar charts
* Persistent data storage using JSON
* Modern UI with blurred background using CustomTkinter

## 🛠️ Technologies Used
* Python
* CustomTkinter (GUI)
* Tkinter & ttk (Table UI)
* Matplotlib (Charts)
* JSON (Data Storage)
* Pillow (Image Processing)

## 💡 Project Overview
This project focuses on building a user-friendly expense tracking system that combines functionality with an improved user interface. It allows users to monitor their spending habits and visualize expenses through graphical representation, making financial tracking simple and effective.

## 🎯 Learning Outcomes
* Strengthened Python programming concepts
* Learned GUI development using CustomTkinter
* Implemented data storage using JSON
* Gained experience in data visualization using Matplotlib
* Improved UI/UX design skills

## ▶️ How to Run
1. Install required libraries:
   ```bash
   pip install customtkinter tkcalendar pillow matplotlib
   ```
2. Run the application:
   ```bash
   python main.py
   ```
